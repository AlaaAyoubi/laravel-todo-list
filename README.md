# Todo List with Laravel

مشروع لإدارة المهام مع تحديد الأولوية (عالي/متوسط/منخفض).

## المميزات
- إضافة مهام جديدة.
- تحديد أولوية المهمة.
- تصفية المهام حسب الأولوية.

## كيفية التنصيب
1. `composer install`
2. انسخ ملف `.env.example` إلى `.env` وعدّل إعدادات قاعدة البيانات.
3. `php artisan migrate`
